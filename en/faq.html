Frequently Asked Questions

Product Code Guide

Every product follows one shared logic. The code tells you the brand, section, category, and item sequence in a single line.

DCBrand: Daban-Cêlê M / WSection: Men / Women TP / BT / ESCategory: Tops / Bottoms / Essentials 001Item sequence number

DC-M-TP-001 = Men / Tops / 1st item

DC-W-BT-002 = Women / Bottoms / 2nd item

DC-W-ES-001 = Women / Essentials / 1st item

When will products officially launch?

The launch timing will be shared through future brand announcements, selected editorial updates, and new drop communication.

Does the website process payments?

No. There is no checkout, payment system, cart flow, banking screen, or order management panel on the site.

Is there a shipping system?

No. The current structure does not include shipping flow, delivery tracking, return forms, or exchange panels.

Are user details collected?

No. The site does not request or store identity details, address data, payment information, or sensitive personal data.

What kind of fabrics are planned?

The focus is on comfortable, flexible, durable fabric groups that support an activewear performance feel.

How does the product code system work?

The code format follows one shared logic: DC-M-TP-001 for men's tops or DC-W-BT-001 for women's bottoms, keeping category and sequence clear at a glance.

How do I get in touch?

Communication happens directly through WhatsApp and phone contact without any on-site user account or request flow.

How will products appear on the site?

As real collections are added, every piece will follow the same card structure: image, product name, code, category, and direct contact.

Will the brand expand beyond Turkey?

The long-term direction aims for a broader community reach; any concrete international steps will be announced later.

Where will products be made?

Production details will be shared clearly and transparently once that structure is finalised.

Can the logo or brand visuals be reused?

No. The Daban-Cêlê name, logo, visual identity, and design language may not be used without written permission.
